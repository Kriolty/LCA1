# LCA1 — Leads Club Australia: Public Marketing Site

**Domain:** leadsclubaustralia.com.au

This is the public-facing marketing website for Leads Club Australia. It contains:

- **Home** — Main landing page with value proposition, lead categories, pricing, and how-it-works sections
- **Solar Leads** — Dedicated vertical landing page for solar energy leads
- **Property Leads** — Dedicated vertical landing page for property investment leads
- **Finance Leads** — Dedicated vertical landing page for finance & loan leads
- **Spa Leads** — Dedicated vertical landing page for spa/wellness leads
- **Competition Entry** — Lead capture competition page

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router v7

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

No environment variables required for the marketing site. Login/Register links redirect to the main app at `app.leadsclubaustralia.com.au`.

## Deployment

Deploy to Vercel, Netlify, or Railway as a static site. No backend required.

```bash
npm run build
# Deploy the /dist folder
```
